    ```python
    def get_connection(db_path: Path) -> sqlite3.Connection:
    ```

    - Return conn

    - Use `with conn:` context manager for atomic DDL

    - CREATE INDEX IF NOT EXISTS idx_commands_namespace ON commands(namespace)
    - CREATE INDEX IF NOT EXISTS idx_commands_created_at ON commands(created_at)

    ```python
    def insert_command(conn: sqlite3.Connection, cmd: Command) -> None:
    ```
    - Use `json.dumps(cmd.allowed_tools)` for the allowed_tools column (per Pitfall 4)

    - Use `with conn:` for atomic insert
    - Parameterized INSERT with `?` placeholders (never f-strings, per anti-patterns)
