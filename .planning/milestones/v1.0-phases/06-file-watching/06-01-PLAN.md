  <action>
    Create src/navigator/watcher_handler.py with:

    1. parse_time_window(window_str: str) -> tuple[time, time]: Split on "-", parse each side with strptime("%H:%M"), return (start, end). Raise ValueError on bad format.

    2. is_within_window(window_str: str | None) -> bool: If None, return True. Parse with parse_time_window. If start <= end (normal day range), check start <= now <= end. If start > end (overnight), check now >= start or now <= end. Use datetime.now().time(). Per D-10, return False (skip silently) outside window.

    3. class SelfTriggerGuard: Per D-07. Has _executing bool (default False) and _lock (threading.Lock). set_executing(bool) and is_executing property, both thread-safe under lock.

    4. class DebouncedHandler(PatternMatchingEventHandler): Per D-05, D-06. Constructor takes callback (Callable[[], None]), debounce_seconds (float), plus PatternMatchingEventHandler kwargs (patterns, ignore_patterns). Has _timer (threading.Timer | None) and _lock (threading.Lock). on_any_event: skip directory modified events, then under lock cancel existing timer and start new one with debounce_seconds delay calling _fire. _fire calls self._callback(). Timer thread is daemon=True.

    5. make_trigger_callback(watcher: Watcher, config: NavigatorConfig, guard: SelfTriggerGuard) -> Callable[[], None]: Returns a closure that:
       a. Checks guard.is_executing -- if True, log debug and return (per D-07)
       b. Checks is_within_window(watcher.active_hours) -- if False, log debug and return (per D-10)
       c. Opens fresh DB connection (thread safety), loads command by name
       d. If command is None or paused, log debug and return
       e. Sets guard.set_executing(True)
       f. Calls execute_command(cmd, config) in try/finally
       g. Sets guard.set_executing(False) in finally block

    6. Write tests/test_watcher_handler.py. For time window tests, monkeypatch datetime.now() to control the clock.
