  <action>
Create `src/navigator/service.py` with the following functions:

1. **`generate_unit_file() -> str`**: Returns the systemd unit file content as a string. Per D-01/D-02/D-03/D-04:
   - Use `shutil.which("navigator")` to resolve the navigator binary path (same pattern as scheduler.py).
   - If `shutil.which` returns None, raise `FileNotFoundError("navigator binary not found on PATH")`.
   - Unit file template (use a Python triple-quoted string, NOT Jinja2):
     ```
     [Unit]
     Description=Navigator Watcher Daemon
     After=network.target

     [Service]
     Type=simple
     ExecStart={navigator_path} daemon
     Restart=on-failure
     RestartSec=5
     Environment=PYTHONUNBUFFERED=1

     [Install]
     WantedBy=default.target
     ```
   - Return the formatted string.

2. **`get_service_path() -> Path`**: Returns `Path.home() / ".config" / "systemd" / "user" / "navigator.service"`. Do NOT use platformdirs here -- systemd user units have a fixed path per D-01.

3. **`install_service(enable_linger: bool = True) -> Path`**: Per D-02/D-06:
   - Call `generate_unit_file()` to get content.
   - Create parent dirs (`mkdir(parents=True, exist_ok=True)`).
   - Write unit file to `get_service_path()`.
   - Run `subprocess.run(["systemctl", "--user", "daemon-reload"], check=True)`.
   - Run `subprocess.run(["systemctl", "--user", "enable", "navigator.service"], check=True)`.
   - If `enable_linger` is True, run `subprocess.run(["loginctl", "enable-linger"], check=False)` (don't fail if loginctl unavailable).
   - Return the service path.

4. **`uninstall_service() -> bool`**:
   - Run `subprocess.run(["systemctl", "--user", "stop", "navigator.service"], check=False)`.
   - Run `subprocess.run(["systemctl", "--user", "disable", "navigator.service"], check=False)`.
   - Delete the unit file if it exists (`get_service_path().unlink(missing_ok=True)`).
   - Run `subprocess.run(["systemctl", "--user", "daemon-reload"], check=True)`.
   - Return True if the file existed before deletion, False otherwise.

5. **`service_control(action: str) -> subprocess.CompletedProcess`**: Per D-07:
   - Validate `action` is one of `("status", "start", "stop", "restart")`, raise `ValueError` otherwise.
   - Run `subprocess.run(["systemctl", "--user", action, "navigator.service"], capture_output=True, text=True)`.
   - Return the CompletedProcess result.

Use `from __future__ import annotations` at top. Use `import shutil` and `import subprocess` from stdlib. Add module docstring following project convention.
