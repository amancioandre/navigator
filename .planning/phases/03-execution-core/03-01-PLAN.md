    - build_clean_env({"API_KEY": "val"}) includes API_KEY in result alongside whitelist vars
    - build_clean_env omits whitelist vars not present in os.environ
    - build_command_args("do thing", ["Read", "Bash"]) returns ["claude", "-p", "do thing", "--print", "--allowedTools", "Read", "--allowedTools", "Bash"]
    - build_command_args("do thing", []) returns ["claude", "-p", "do thing", "--print"] with no --allowedTools
    - execute_command calls subprocess.run with env=build_clean_env result, cwd=str(cmd.environment), capture_output=True, text=True
    - execute_command raises FileNotFoundError with helpful message if claude not on PATH
    - execute_command raises FileNotFoundError with helpful message if cwd does not exist
    - Secret values never appear in the args list passed to subprocess.run
  </behavior>
  <action>
    1. Create `src/navigator/executor.py` with:
       - Module docstring: "Navigator execution engine -- launches Claude Code subprocesses with environment isolation."
       - `from __future__ import annotations`
       - `import logging`, `import os`, `import shutil`, `import subprocess`, `from pathlib import Path`
       - `from navigator.models import Command`
       - `logger = logging.getLogger(__name__)`
       - Constant `ENV_WHITELIST: tuple[str, ...] = ("PATH", "HOME", "LANG", "TERM", "SHELL")` (per D-09, D-10, D-11)
       - Function `build_clean_env(secrets: dict[str, str] | None = None) -> dict[str, str]`:
         - Start with empty dict
         - Copy only ENV_WHITELIST vars from os.environ (skip if not present) (per D-09)
         - Merge secrets if provided
         - Return env dict
       - Function `build_command_args(prompt: str, allowed_tools: list[str]) -> list[str]`:
         - Build list: `["claude", "-p", prompt, "--print"]` (per D-02)
         - For each tool in allowed_tools: extend with `["--allowedTools", tool]` (per D-03)
         - Do NOT use `--dangerously-skip-permissions` (per D-04)
         - Return args list
       - Function `execute_command(cmd: Command) -> subprocess.CompletedProcess[str]`:
         - Validate claude exists: `if shutil.which("claude") is None: raise FileNotFoundError("claude CLI not found on PATH. Install Claude Code or verify PATH.")`
         - Validate cwd exists: `if not cmd.environment.is_dir(): raise FileNotFoundError(f"Working directory does not exist: {cmd.environment}")`
         - `from navigator.secrets import load_secrets` (lazy import)
         - `secrets = load_secrets(cmd.secrets)`
         - `env = build_clean_env(secrets)`
         - `args = build_command_args(cmd.prompt, cmd.allowed_tools)`
         - Log info: `"Executing command '%s' in %s", cmd.name, cmd.environment` (no secret values in log, per EXEC-03)
         - `result = subprocess.run(args, env=env, cwd=str(cmd.environment), capture_output=True, text=True)` (per D-01, EXEC-07, EXEC-08)
         - Log info: `"Command '%s' exited with code %d", cmd.name, result.returncode`
         - Return result
    2. Create `tests/test_executor.py` with tests using `unittest.mock.patch`:
       - `test_build_clean_env_whitelist_only`: Monkeypatch os.environ to known values, assert only ENV_WHITELIST keys appear in result.
       - `test_build_clean_env_with_secrets`: Verify secrets are merged into env dict.
       - `test_build_clean_env_skips_missing_vars`: Set os.environ without LANG, verify LANG not in result.
       - `test_build_command_args_with_tools`: Assert `build_command_args("do thing", ["Read", "Bash"])` == `["claude", "-p", "do thing", "--print", "--allowedTools", "Read", "--allowedTools", "Bash"]`.
       - `test_build_command_args_no_tools`: Assert `build_command_args("do thing", [])` == `["claude", "-p", "do thing", "--print"]`.
       - `test_execute_command_passes_env_and_cwd`: Mock subprocess.run and shutil.which, create a Command with environment pointing to tmp_path, call execute_command, assert subprocess.run was called with `env` containing only whitelist keys, `cwd=str(tmp_path)`, `capture_output=True`, `text=True`.
       - `test_execute_command_claude_not_found`: Mock shutil.which to return None, assert FileNotFoundError raised with "claude CLI not found" message.
       - `test_execute_command_cwd_not_exist`: Create Command with nonexistent environment path, mock shutil.which to return "/usr/bin/claude", assert FileNotFoundError raised with "Working directory does not exist" message.
       - `test_secrets_not_in_args`: Mock subprocess.run and shutil.which, create Command with secrets file, verify the args list passed to subprocess.run contains no secret values (only "claude", "-p", prompt, "--print", "--allowedTools", tool names).
       - `test_no_dangerously_skip_permissions`: Verify "--dangerously-skip-permissions" never appears in build_command_args output.
    3. Run `uv run pytest tests/test_executor.py -x` -- all tests must pass.
    4. Run full suite: `uv run pytest -x` -- ensure no regressions.
