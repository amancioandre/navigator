## Tech Debt

### Phase 10: Operational Polish
- `exec` live execution path does not emit JSON when `--output json` is set — scripts/agents get Rich text instead of structured data
- Mutation commands (register, update, delete, pause, resume, chain, watch, schedule, namespace create/delete) do not emit JSON — lower priority since these are less frequently scripted

**Impact:** Low-medium. The most common scripted operation (`exec`) should have JSON output for full Claude Code agent integration. List/show/logs/doctor already have JSON output which covers the most common introspection use cases.
