**Module-level constant:**
- `COMMENT_PREFIX = "navigator"` -- used to build comment tags per D-01

**CrontabManager.__init__(self, lock_path: Path):**
- Stores `lock_path` (will be `{config.data_dir}/crontab.lock` per D-05)

**CrontabManager._make_comment(self, command_name: str) -> str:**
- Returns `f"{COMMENT_PREFIX}:{command_name}"` per D-01

**CrontabManager._resolve_navigator_path(self) -> str:**
- Uses `shutil.which("navigator")` per SCHED-05
- Raises `FileNotFoundError` with clear message if None per Pitfall 5

**CrontabManager._lock(self):**
- `@contextmanager` using `fcntl.flock` with `LOCK_EX | LOCK_NB` per D-05
- Retry loop with `time.monotonic()` deadline of 10 seconds per D-06
- On timeout: raise `TimeoutError` with message about another navigator process per D-06
- Creates lock_path parent directory if needed
- Lock covers entire read-modify-write per D-07
- Unlock and close fd in finally block

**CrontabManager.schedule(self, command_name: str, cron_expr: str) -> None:**
- Resolve navigator path first (before lock)
- Inside `_lock()` context: `CronTab(user=True)` per D-03
- Remove existing entries with `find_comment()` (wrapped in `list()` per Pitfall 4) for idempotent update
- Create new entry: `cron.new(command=f"{nav_path} exec {command_name}", comment=self._make_comment(command_name))`
- `job.setall(cron_expr)` -- catch both `KeyError` and `ValueError`, re-raise as `ValueError` with clean message per D-04 / Pitfall 2
- Check `job.is_valid()` -- raise `ValueError` if invalid per D-04
- `cron.write()` per D-03
- Call `_verify_entry(command_name)` after write, raise `RuntimeError` if verification fails per D-11

**CrontabManager.unschedule(self, command_name: str) -> bool:**
- Inside `_lock()`: `CronTab(user=True)`, `find_comment()`, `cron.remove(job)` for each, `cron.write()`
- Returns True if entries were found and removed, False if nothing to remove per SCHED-03

**CrontabManager.list_schedules(self) -> list[dict[str, str]]:**
- `CronTab(user=True)`, iterate all jobs
- Filter where `job.comment.startswith(f"{COMMENT_PREFIX}:")`
- Return list of dicts: `{"command": cmd_name, "schedule": str(job.slices), "enabled": str(job.is_enabled()), "cron_command": str(job.command)}`
- No lock needed (read-only, single atomic read)

**CrontabManager._verify_entry(self, command_name: str) -> bool:**
- Re-read with `CronTab(user=True)`, `find_comment()`, return `len(found) > 0` per D-11

**Fixture pattern:**
```python
@pytest.fixture()
def crontab_env(tmp_path, monkeypatch):
    """Set up mock crontab environment for scheduler tests."""
    tab_file = tmp_path / "crontab"
    tab_file.write_text("")
    lock_path = tmp_path / "crontab.lock"

    # Monkeypatch CronTab in scheduler module to use tabfile
    from crontab import CronTab as RealCronTab
    monkeypatch.setattr(
        "navigator.scheduler.CronTab",
        lambda **kw: RealCronTab(tabfile=str(tab_file)),
    )
    monkeypatch.setattr(
        "navigator.scheduler.shutil",
        type("shutil", (), {"which": staticmethod(lambda cmd: "/usr/local/bin/navigator")}),
    )

    from navigator.scheduler import CrontabManager
    manager = CrontabManager(lock_path=lock_path)
    return {"manager": manager, "tab_file": tab_file, "lock_path": lock_path, "tmp_path": tmp_path}
```
