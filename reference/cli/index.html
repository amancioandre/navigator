    CLI Reference

    Complete reference for the navigator command-line interface.

    navigator

    Autonomous task orchestrator for Claude Code sessions.

    Usage:

    navigator [OPTIONS] COMMAND [ARGS]...

    Options:

    Name Type Description Default
    --version boolean Show version and exit. None
    --output text Output format: text or json None
    --install-completion boolean Install completion for the current shell. None
    --show-completion boolean Show completion for the current shell, to copy it or customize the installation. None
    --help boolean Show this message and exit. False

    Subcommands

    • chain: Chain commands to run sequentially after each other.
    • daemon: Run the watcher daemon in foreground (for systemd).
    • delete: Delete a registered command.
    • doctor: Verify system health and diagnose configuration issues.
    • exec: Execute a registered command.
    • install-service: Generate and install the systemd user service.
    • list: List all registered commands.
    • logs: View execution logs for a command.
    • namespace: Manage namespaces for organizing commands into groups.
    • pause: Pause a registered command.
    • register: Register a new command.
    • resume: Resume a paused command.
    • schedule: Schedule a command with a cron expression.
    • service: Manage the Navigator systemd service (start, stop, restart, status).
    • show: Show full details of a registered command.
    • uninstall-service: Remove the systemd user service.
    • update: Update fields of a registered command.
    • watch: Register a file watcher for a command.

    chain

    Chain commands to run sequentially after each other.

    Usage:

    navigator chain [OPTIONS] COMMAND

    Options:

    Name Type Description Default
    --next text Next command to chain (supports namespace:command) None
    --show boolean Show the chain starting from this command False
    --remove boolean Remove chain link from this command False
    --on-failure text Failure mode: 'stop' (default) or 'continue' None
    --help boolean Show this message and exit. False
