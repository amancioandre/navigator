  <action>
    Add run_daemon(config: NavigatorConfig) -> None function to watcher.py:

    1. Import Observer from watchdog.observers, DebouncedHandler and make_trigger_callback and SelfTriggerGuard from navigator.watcher_handler
    2. Create a single Observer instance
    3. Load all active watchers via WatcherManager(config).get_active_watchers()
    4. If no active watchers, print message and return (do not start empty observer)
    5. For each active watcher:
       a. Create a SelfTriggerGuard instance (one per watcher)
       b. Create callback via make_trigger_callback(watcher, config, guard)
       c. Create DebouncedHandler with callback=callback, debounce_seconds=watcher.debounce_ms / 1000.0, patterns=watcher.patterns, ignore_patterns=watcher.ignore_patterns
       d. Call observer.schedule(handler, str(watcher.watch_path), recursive=watcher.recursive)
       e. Log info: "Watching {watcher.watch_path} for command '{watcher.command_name}'"
    6. Call observer.start()
    7. Log info: "Watcher daemon started. Monitoring {len(watchers)} watcher(s). Press Ctrl+C to stop."
    8. Block in while loop: while observer.is_alive(): observer.join(timeout=1)
    9. Wrap in try/except KeyboardInterrupt: observer.stop()
    10. observer.join() after stop

    This is a foreground process per discretion decision. Phase 9 adds systemd.
  </action>
